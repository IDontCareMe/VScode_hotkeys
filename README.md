# VScode_hotkeys for Windows
VS Code hotkeys/Горячие клавиши VS Code

#### File->Preferences->Settings:
```
Ctrl + ,
```

#### View->Command Palette...:
```
Ctrl + Shift + P
```

#### View->Explorer:
```
Ctrl + Shift + E
```

#### View->Extencions:
```
Ctrl + Shift + X
```

#### Terminal->New Terminal:
```
Ctrl + Shift + `
```
